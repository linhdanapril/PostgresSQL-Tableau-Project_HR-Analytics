# 📊 HR Analytics Dashboard

Analyzing HR data using **Tableau** and **PostgreSQL**, focused on employee attrition and related factors.

## 📁 Files

| File | Description |
|---|---|
| `Tableau_HR.twb` | Tableau dashboard |
| `Postgres_HR.sql` | PostgreSQL queries |

## 📌 Dashboard Overview

- **1,470** employees · **16.12%** attrition rate · **1,233** active · avg. age **37**
- Attrition by department, age group, gender, and education field
- Job satisfaction rating by job role

## 🛠️ Requirements

- Tableau Desktop or [Tableau Public](https://public.tableau.com/) (free)
- PostgreSQL
